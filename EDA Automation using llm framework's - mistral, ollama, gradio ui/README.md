# 🤖 EDA Automation using LLM Frameworks

### *Mistral · Ollama · Gradio UI*

This project showcases an **automated Exploratory Data Analysis (EDA) pipeline** powered by **Large Language Models (LLMs)** such as **Mistral** and **Ollama**, wrapped inside an intuitive **Gradio-based web interface**.

The goal is simple but powerful:
👉 *Upload a dataset → get statistical summaries → receive human-like insights generated by an LLM.*

Perfect for **data analysts, data scientists, and ML engineers** who want faster insights with minimal manual effort.

## 🧠 Key Capabilities

* 📂 Upload CSV datasets directly from the browser
* 📊 Automatic statistical analysis using **Pandas**
* 🧠 LLM-powered interpretation of EDA results
* 🗣️ Converts raw numbers into **natural language insights**
* 🎛️ Clean, interactive **Gradio UI**
* 🔐 Supports **local LLM execution** using Ollama (no cloud dependency)

---

## 🛠️ Tech Stack

* 🧠 **Mistral** – Generates intelligent, contextual EDA insights
* 🧠 **Ollama** – Runs LLMs locally for privacy & speed
* 🎛️ **Gradio** – Interactive and lightweight web interface
* 📊 **Pandas** – Data loading, cleaning, and summary statistics
* 🐍 **Python** – Core application logic

## ⚙️ How It Works

1. 📂 User uploads a CSV file via the Gradio interface
2. 📊 The dataset is analyzed using `pandas.describe()` and basic profiling
3. 🧠 The statistical summary is passed to **Mistral / Ollama**
4. 🗣️ The LLM generates **human-readable insights** explaining patterns, distributions, and anomalies
