## 🛠️ Getting Started

### ✅ Option 1: GitHub Codespaces (Recommended)

* Zero setup
* Auto environment configuration
* App launches automatically on port `8501`

Steps:

1. Click **Code → Codespaces**
2. Create a new Codespace
3. Wait ~1–2 minutes
4. Start chatting

---

### 💻 Option 2: Local Setup

```bash
git clone https://github.com/your-username/ai-nexus-pro.git
cd ai-nexus-pro
pip install -r requirements.txt
streamlit run app.py
```

## ⚙️ Configuration

### Hugging Face Setup

```python
self.huggingface_token = "YOUR_HF_TOKEN"
self.available_models = [
    "HuggingFace AI Pro",
    "Advanced Demo AI"
]
```

Store tokens securely using environment variables for production use.
