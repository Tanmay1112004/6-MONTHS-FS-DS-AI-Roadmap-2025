## 🗺️ Learning Roadmap

### 🧱 1️⃣ Data Manipulation & Operations (`sql_practice.sql`)

A hands-on SQL script built for execution in **MySQL** or **PostgreSQL** environments.

Covered Concepts:

* ✅ **CRUD Operations** – `CREATE`, `SELECT`, `UPDATE`, `DELETE`
* ✅ **Filtering & Conditions** – `WHERE`, `BETWEEN`, `IN`, `LIKE`
* ✅ **Logical Operators** – `AND`, `OR`, `NOT`
* ✅ **Comparison Operators** – `=`, `>`, `<`, `!=`
* ✅ **Arithmetic Expressions** – Calculations on data fields
* ✅ **Date & Numeric Functions**
* ✅ **Boolean Logic for reporting**

This file is designed to be executed step-by-step for active learning.
