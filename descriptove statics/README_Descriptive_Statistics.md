# 📊 Household Descriptive Statistics Analysis

### Turning Raw Data into Financial Insights

An in-depth **Exploratory Data Analysis (EDA)** project that examines household financial behavior using descriptive statistics and visualization techniques.

This study transforms a small real-world dataset into meaningful insights about **income patterns, spending behavior, and household composition**.

---

## 🧠 Project Overview

Understanding household finances is essential for policymakers, financial institutions, and market analysts.

This project analyzes key indicators to evaluate:

✅ Spending patterns
✅ Financial variability
✅ Income–expense relationships
✅ Household stability factors

---

## 📂 Dataset Summary

| Attribute  | Details                                                     |
| ---------- | ----------------------------------------------------------- |
| Records    | 50 households                                               |
| Features   | 7 financial & demographic variables                         |
| Key Fields | Income, Expenses, EMI/Rent, Household Size, Earning Members |
| Objective  | Identify trends, central tendencies, and variability        |

---

## 🔍 Key Findings

### 💰 Central Tendency — Monthly Expenses

📈 **Mean:** ₹18,818
📊 **Median:** ₹15,500
🔥 **Mode:** ₹25,000

➡️ The higher mean compared to median suggests the presence of high-spending households, indicating skewness and potential outliers.

---

### 📉 Variability & Distribution

📏 **Interquartile Range (IQR):** ₹15,000
📊 High dispersion across households
⚖️ Coefficient of Variation used for relative comparison

➡️ Spending behavior varies significantly due to differences in income levels, family size, and financial obligations.
